Escape Game
Synopsis :
Vous avez été kidnappé par un groupe de personnes mal intentionné lors d'une soirée publique un peu trop arrosée.
Par chance, ces personnes ont décidés de ne pas vous tuer et de vous laisser indemne.
Vous n'avez aucun souvenir du moment où vous êtes parti, ni d'où vous êtes à l'heure actuelle.
Votre but est de sortir de ce lieu, en trouvant un maximum d'indices pour vous échapper. Bonne chance!
C'est parti !
